# Seoul-Subway-FreeRide-Analysis
# 🚇 지하철 노인 무임승차 수요 예측 및 요금 정책 최적화

## 💡 프로젝트 배경 및 목표
* **배경:** 고령화 사회 진입에 따른 무임승차 수요 증가와 지하철 재정 적자 문제 대두
* **목표:** 2050년까지의 장기 수요를 예측하고, 사회순후생(NSB)을 극대화하는 다구간 요금 정책 시뮬레이션 및 대안 도출

## 🛠 기술 스택
* **Language & Data:** Python, Pandas, Numpy, SQL
* **Machine Learning:** Scikit-learn, XGBoost, LightGBM
* **Optimization:** Grid Search 적용

## 📊 데이터 파이프라인 및 전처리
* **사용 데이터:** 서울시 열린데이터광장 승하차 인원, 기상청 관측 자료, 25개 자치구 미세먼지 데이터
* **전처리:**
  * 다년도 데이터 병합 중 발생한 인코딩 불일치 문제 해결
  * 결측치 보간(Imputation) 파이프라인 구축

## 🧠 모델링 및 최적화
1. **수요 예측 (Ensemble Model):**
   * 선형 회귀 모델의 한계를 극복하기 위해 비선형 트리 기반 모델 도입
   * XGBoost와 LightGBM을 50:50으로 결합한 Voting 앙상블 모델 구축
   * **성과:** 예측 오차율(MAPE) 7.56% 달성
2. **정책 시뮬레이션:**
   * 로지스틱 함수와 Grid Search 알고리즘을 활용하여 최적의 요금 부과 로직 탐색

## 🚀 결론 및 비즈니스 임팩트
* 단순히 예측 정확도를 높이는 것을 넘어, 예측된 데이터를 바탕으로 실제 정책 결정에 활용할 수 있는 최적화된 요금제 대안 제시.
